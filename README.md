> MiniProgramSDK是一个PHP的小程序开发包。封装微信小程序后端相关接口，比如解密信息（用户信息、手机号）、获取小程序码等。

### 注意

- 由于精力有限，v2版本并未发布，目前准备重写v2版本。
- v1版本可正常使用。[v1文档](https://hillpy.github.io/MiniProgramSDK/v1/)
